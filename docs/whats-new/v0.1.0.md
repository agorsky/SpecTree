# Version 0.1.0

**Release Date:** February 2026

## Overview

Initial release of SpecTree - A Linear-inspired project management and issue tracking platform designed for AI-assisted software development workflows.

## Features

### Core Platform
- **Team-based collaboration** with role-based access control (admin, member, guest)
- **Personal workspace** for private projects and tasks
- **Hierarchical work organization** (Epics → Features → Tasks)
- **Customizable workflows** with team-specific and personal status definitions
- **Multi-database support** (PostgreSQL, SQL Server, SQLite)
- **Epic requests** workflow for proposing, voting on, commenting on, and approving new initiatives
- **Epics grouped by team** on the epics page for organized navigation
- **Epic scope filtering** (My Epics, All, By Team, By User) with localStorage persistence
- **Version tracking** with in-app "What's New" changelog

### AI Integration
- **MCP (Model Context Protocol) server** for seamless AI assistant integration
- **Structured descriptions** with AI-friendly extractable sections
- **Code context tracking** linking work items to files, functions, git branches, commits, and PRs
- **AI session management** with handoff capabilities between sessions
- **Real-time session progress monitoring** with live status updates
- **Decision logging** to preserve implementation rationale
- **Validation checklists** for executable acceptance criteria
- **Planner agent integration** with epic requests via `--from-request` flag

### Developer Experience
- **Implementation plan templates** for standardized work structures
- **Epic requests** workflow for proposing new initiatives with reactions and comments
- **Execution planning** with dependency management and parallel execution support
- **Progress tracking** with comprehensive summary dashboards
- **Admin activity dashboard** with filtering by scope (self, team, user), drill-down views, and timezone-aware date handling
- **Changelog tracking** at entity level with field-level change details
- **Reusable scope selector component** shared across dashboard and epics pages

### API & Architecture
- **RESTful API** with comprehensive endpoint coverage
- **Server-Sent Events (SSE)** for real-time updates
- **Secure authentication** with session management and CSRF protection
- **Monorepo structure** with TypeScript, Fastify, and React
- **Deployment ready** with Azure Web App support and GitHub Actions CI/CD

## Technical Details

### Technology Stack
- **Backend:** Node.js 20+, TypeScript, Fastify
- **Frontend:** React, TypeScript, Vite
- **Database:** SQLite (dev), PostgreSQL/SQL Server (production)
- **Build System:** Turborepo with pnpm workspaces
- **Testing:** Vitest
- **Deployment:** Azure Web Apps, Docker support

### Security Features
- Session-based authentication with secure cookie handling
- CSRF protection for state-changing operations
- Password hashing with bcrypt
- Role-based access control (RBAC)
- Invite-only team collaboration model

### Developer Tools
- ESLint and Prettier for code quality
- Git hooks for pre-commit validation
- Automated schema migrations
- Comprehensive documentation
- MCP server for AI integration with Claude and other assistants

## Known Limitations

- Email delivery not yet implemented (invitations stored but not sent)
- Search functionality limited to title and description fields
- No file attachment support
- Single-region deployment only

## Migration Notes

This is the initial release. No migrations required.
